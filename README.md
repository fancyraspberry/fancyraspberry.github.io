# Mohamed Jad El Hajjar — Portfolio

Personal portfolio website built with vanilla HTML, CSS and JavaScript.

## Structure

```
portfolio/
├── index.html              # Main page
├── css/
│   └── style.css           # All styles
├── js/
│   └── main.js             # Scroll reveal, language toggle
├── assets/
│   └── CV_EL_HAJJAR_Mohamed_Jad.pdf
└── README.md
```

## Features

- Bilingual (FR / EN) with a toggle
- Downloadable CV
- Scroll-reveal animations
- Fully responsive
- No dependencies — pure HTML/CSS/JS

## Deploy

Works with GitHub Pages out of the box. Push to `main`, enable Pages from the repo settings.
