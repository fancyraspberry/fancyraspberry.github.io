Application de Gestion Java

Dans le cadre du second semestre, j'ai développé avec mon groupe une application Java destinée à la gestion d'inscriptions étudiantes. Le projet couvrait à la fois la conception d'une interface graphique avec Java Swing et la gestion des données via une base relationnelle. L'objectif était de produire une application fonctionnelle, structurée et ergonomique — pas juste du code qui tourne.

  • Implémentation complète back-end et front-end via Java Swing
  • Architecture orientée objet avec séparation claire des responsabilités
  • Travail approfondi sur l'ergonomie et l'interaction humain-machine
  • Gestion des données et contrôles de saisie robustes

Ce projet m'a permis de confronter mes compétences à un développement applicatif complet — de la logique métier à l'interface utilisateur — dans un contexte d'équipe avec des contraintes réelles.

Administration Système & BDD

Ce projet s'est déroulé en deux phases. La première, individuelle, consistait à rencontrer un interlocuteur réel, analyser ses besoins et produire un modèle conceptuel de données (MCD), un dictionnaire de données et une synthèse. La seconde phase, en binôme, portait sur l'implémentation technique via SQLite.

  • Conception du MCD, du schéma relationnel et des règles de gestion
  • Scripts SQL de création, d'insertion et d'interrogation des données
  • Installation et configuration de Linux sur Raspberry Pi
  • Utilisation d'un générateur de données pour simuler un environnement réaliste
  • Rédaction d'un rapport technique structuré

Ce projet m'a appris à travailler sur un cycle complet de base de données — de l'analyse des besoins jusqu'à la mise en œuvre physique — en documentant chaque choix technique de manière rigoureuse.

Développement Web & Algorithmique

Dans le cadre du module CWAD (Communication Web et Accessibilité Digitale), nous avons conçu un site web statique autour d'un parc de loisirs médiéval fictif — dragons, princesses, tournois et forteresses. L'idée était de bâtir un univers cohérent visuellement et de le rendre accessible selon les standards du web.

  • Création d'une identité visuelle complète (palette, typographie, logo)
  • Développement d'un site statique HTML/CSS respectant les standards d'accessibilité
  • Contenu structuré et narratif autour du thème médiéval
  • Rédaction d'un rapport technique sur les choix graphiques et structurels
  • Soutenance orale avec maquette fonctionnelle

En parallèle, j'ai programmé un jeu vidéo en C++ en binôme pour comparer différentes approches algorithmiques sur un même problème — un exercice qui m'a fait réaliser combien les choix d'implémentation peuvent changer radicalement les performances.

Code Game Jam 2025

Lors de la Code Game Jam 2025, notre équipe de 5 a eu 26 heures pour concevoir et livrer un jeu vidéo complet, de A à Z, autour d'un thème révélé au lancement. Ce format marathon nous a forcés à couvrir toutes les étapes en accéléré : brainstorming, répartition des tâches, développement, tests et présentation finale.

Ce sprint créatif m'a montré ce que ça veut dire de livrer sous pression — prendre des décisions rapides, s'adapter en temps réel, et trouver l'équilibre entre ambition et faisabilité. Une des expériences les plus stimulantes de l'année.
